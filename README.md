# OSI: OS Information

A Windows Operating System Information project



## Authors

* [@remusrigo](https://github.com/RemusRigo)
* [@remusrigo](https://sourceforge.net/u/remusrigo/profile)



## Installation

You must install .NET Runtime 9 x64 and then just run Whisperer.exe



## Features

* Display information:

  * Operating System
  * Software\Environment Variables
  * Software\Users
  * Hardware\Disk Drive
